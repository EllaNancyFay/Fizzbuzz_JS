Fizzbuzz
==================

###Creating a simple TDD Fizzbuzz Kata in JavaScript

####Technologies Used:

  - JavaScript
  - Jasmine

#####To do list:

How to use
----------
Clone the repository:
```shell
$ git clone github.com/EllaNancyFay/Fizzbuzz_JS.git
```

Change into the directory:
```shell
$ cd fizzbuzz
```

To run tests:
```shell
$ open SpecRunner.html
```
